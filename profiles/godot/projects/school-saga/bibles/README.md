# School Saga Bibles Index

This directory is the project-specific home for authoritative product and narrative material, including design, world, character, and story bibles.

Bible documents are product truth, not engineering suggestions. Agents receive only the sections relevant to the assigned issue.

Rules:

- do not infer missing canon;
- do not reconcile conflicting canon without human direction;
- record document paths and commit SHAs in the context package;
- update Bible material only when the issue explicitly authorizes a product decision;
- distinguish aspirational design from implemented behavior when the source does so.

The initial profile provides this index only. Authoritative School Saga Bible content must be added by the project owners.
