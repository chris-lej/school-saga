# School Saga ADR Index

Architecture Decision Records capture durable project-specific technical decisions.

Each ADR should include:

- status;
- context;
- decision;
- consequences;
- affected compatibility surfaces;
- superseded or related ADRs.

Agents may apply only accepted ADRs included in their context package. Proposed, rejected, or superseded ADRs are not governing policy unless a human explicitly directs otherwise.
