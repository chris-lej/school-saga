# School Saga Architecture Index

This directory contains project-specific system architecture supplied to agents through bounded context packages.

Each architecture document should define:

- subsystem mission and ownership;
- public interfaces and compatibility surfaces;
- dependencies and prohibited dependencies;
- persistence, scene, resource, and signal contracts;
- relevant validation expectations;
- links to governing ADRs.

Architecture documents are included only when relevant to an issue. This index does not authorize agents to infer undocumented systems.
