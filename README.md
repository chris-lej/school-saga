# School Saga Autonomous Engineering Organization

This repository defines and implements the autonomous software engineering pipeline used to build School Saga.

School Saga is the first concrete implementation of a broader organizational model built around:

- explicit agent contracts;
- GitHub-driven workflow state;
- structured handoff artifacts;
- objective validation;
- independent code review;
- deterministic recovery and retry behavior;
- project-specific context packaged through profiles.

## Current phase

Phase 1: Foundation.

The repository begins with product, architecture, workflow, and organizational documentation before production agent code is introduced.

## Planned phases

1. Foundation
2. Organization architecture
3. Agent contracts
4. School Saga project profile
5. Platform implementation
6. End-to-end validation
7. Generalization
